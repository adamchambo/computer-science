# Computer Science

A practical learning repository covering fundamental computer science concepts through notes, examples, and implementations in multiple programming languages.

The repository currently starts with Python, but future implementations will use a range of languages to explore how the same concepts work across different ecosystems.

## Topics

- Data structures
- Algorithms
- Complexity analysis
- Design patterns
- Software architecture
- Databases
- Networking
- Operating systems
- Security
- System design
- Software testing
- DevOps

Each topic has its own directory and may include a dedicated README with more specific explanations, complexity analysis, implementations, and exercises.

## Under the hood

Concepts are grouped by subject rather than programming language. A concept such as a linked list therefore lives under `data-structures/linked-list/`, with language-specific implementations kept alongside one another as they are added. This makes it easier to compare how the same underlying idea is expressed in different languages.

## Status

This repository is a work in progress. Python is the first implementation language, with more languages and concepts to be added over time.
